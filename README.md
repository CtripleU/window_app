# Window App

A new Flutter project.

## Getting Started

- Install Flutter
+ Install Android Studio along with an emulator
- Clone the repository
- Open it in Android studio
- Set-up an emulator of screen size Pixel 6
- Open emulator
- Run the command `flutter run` in terminal
